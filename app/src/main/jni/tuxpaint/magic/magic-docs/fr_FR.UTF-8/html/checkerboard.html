Outil 'Magie' de Tux Paint : Damier

Groupe: Décorations des images

Auteur : Bill Kendrick <bill@newbreedsoftware.com>

Couvre entièrement le canevas avec un damier en utilisant la couleur sélectionnée. Balayez pour changer la taille des carrés.

Cet outil permet de travailler en mode à main levée.
